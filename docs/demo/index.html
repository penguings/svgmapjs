svgmapjs demo (this fork)

2種類のデモを用意しています（コンテナSVG方式）。 ブラウザのセキュリティ制約のため、file:// 直開きではなく HTTP 配信で開いてください。

  • Basic (extensions なし)
    最小構成（SvgMap コアのみ）
  • Full (extensions 全部)
    Layer UI / WebApp handler / Custom layers / Authoring / Cesium wrapper

例: リポジトリ直下で npx http-server -c-1 . → /docs/demo/
